# AfricaThrift - 非洲二手服装出口外贸站设计规范

## 1. 项目概述

### 1.1 项目背景
- **定位**: B2B外贸独立站，面向非洲批发商、零售商、二手服装采购商
- **核心业务**: 出口二手衣服（Used Clothing/Balanga）至非洲市场
- **目标市场**: 西非（尼日利亚、加纳、喀麦隆）、东非（肯尼亚、坦桑尼亚）、南非
- **核心竞争力**: 货源稳定、价格优势、质检保障、物流直达

### 1.2 品牌命名
| 方案 | 名称 | 域名建议 |
|------|------|----------|
| 方案A | **AfriThrift** | afrithrift.com |
| 方案B | **Kanga Trade** | kangatrade.com |
| 方案C | **Used2Africa** | used2africa.com |

**推荐**: AfriThrift - 融合 Africa + Thrift，体现非洲市场定位与二手服装属性

### 1.3 域名建议
- 主域名: `afrithrift.com`
- 非洲区域域名: `afrithrift.ng` (尼日利亚), `afrithrift.gh` (加纳)
- 备选: `usedclothesafrica.com`

## 2. 市场洞察

### 2.1 非洲二手服装市场特点
| 特征 | 说明 |
|------|------|
| **市场体量** | 非洲二手服装年进口额超15亿美元 |
| **主力产品** | Bales（打包货，每包约45-50kg）、零售散货 |
| **价格敏感** | 采购商对价格极为敏感，追求高性价比 |
| **质量要求** | 分拣质量稳定，不接受严重损坏品 |
| **物流依赖** | 海运为主，周期长（30-45天），清关能力重要 |
| **移动优先** | 70%用户通过手机访问，WhatsApp沟通频繁 |

### 2.2 目标客户画像
| 客户类型 | 特征 | 核心需求 |
|----------|------|----------|
| 批发商 | 每月采购50-200包 | 价格、货量稳定性 |
| 零售商 | 每周采购10-30包 | 质量稳定性、起订量低 |
| 二手店主 | 定期补货 | 品类丰富、款式多样 |
| 电商卖家 | 线上销售 | 物流时效、跟踪服务 |

### 2.3 竞争格局
- **主要竞争对手**: Kikuyu Vintage, Africa Clothing Importers, UsedClothes Africa
- **差异化机会**: 
  - 透明报价系统
  - 在线验货服务
  - 分期付款支持
  - 区块链溯源

- 卡片: 12px
- 输入框: 8px
- 标签: 4px

### 3.5 阴影系统
```
阴影-小: 0 1px 3px rgba(0,0,0,0.12)
阴影-中: 0 4px 6px rgba(0,0,0,0.1)
阴影-大: 0 10px 25px rgba(0,0,0,0.15)
阴影-悬浮: 0 15px 35px rgba(0,0,0,0.2)

使用原则:
- 卡片默认: 阴影-小
- 卡片悬浮: 阴影-中 + 上移4px
- 弹窗: 阴影-大
- 按钮悬浮: 阴影-小 + 颜色加深10%
```

### 3.6 动效规范
```
过渡时长:
- 快速: 150ms (微交互)
- 常规: 250ms (页面元素)
- 缓慢: 400ms (页面切换)

缓动函数:
- 默认: cubic-bezier(0.4, 0, 0.2, 1)
- 进入: cubic-bezier(0, 0, 0.2, 1)
- 退出: cubic-bezier(0.4, 0, 1, 1)
- 弹性: cubic-bezier(0.68, -0.55, 0.265, 1.55)

动画效果:
- 页面加载: 淡入 + 上移20px
- 按钮点击: 缩放0.98
- 图片悬浮: 放大1.05
- 数字滚动: 计数器动画
```
